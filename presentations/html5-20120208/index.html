History

  • Invented 20 years ago ♥ Tim Berners-Lee
  • Over 10 years since HTML 4 (1999)
  • The work on HTML stopped ☄
  • XHTML was the way forward, XHTML 1.0 Completed 2000
  • XHTML 2.0 wasn't backwards compatible with existing content ☠
  • WHATWG formed in 2004 (Opera, Mozilla and later on Apple) ☮
  • Created Web Applications 1.0
  • 2007 the work on HTML restarted with a new HTML WG
  • Started to adopt the Web Applications 1.0 spec and call it HTML5 ☺
  • WHATWG says HTML is the new HTML5 "the specification is now a living document"
  • HTML is still being developed ♥
